=== Income Statement Query Plan ===
Pendapatan:
  FROM orders WHERE is_paid = true AND created_at BETWEEN [$s, $e]
  SUM(total_amount)
  +
  FROM unexpected_transactions WHERE jenis = 'pemasukan' AND created_at BETWEEN [$s, $e]
  SUM(nominal)

HPP:
  FROM ingredient_batches WHERE received_at BETWEEN [$s, $e]
  SELECT SUM(quantity * cost_per_unit)

Laba Kotor = Pendapatan - HPP

Beban Operasional:
  FROM expenses WHERE date BETWEEN [$s, $e]
  SUM(amount)

Beban Tak Terduga:
  FROM unexpected_transactions WHERE jenis = 'pengeluaran' AND created_at BETWEEN [$s, $e]
  SUM(nominal)

Laba/Rugi Bersih = Laba Kotor - Beban Operasional - Beban Tak Terduga

=== Cash Flow Query Plan ===
Arus Kas Masuk:
  Pendapatan (same as Income Statement)
  +
  FROM receivables WHERE status IN ('paid', 'partial') AND updated_at BETWEEN [$s, $e]
  SUM(paid_amount)

Arus Kas Keluar = Beban Operasional + HPP + Beban Tak Terduga

Arus Kas Bersih = Arus Kas Masuk - Arus Kas Keluar

Saldo Awal = 0
Saldo Akhir = Arus Kas Bersih
